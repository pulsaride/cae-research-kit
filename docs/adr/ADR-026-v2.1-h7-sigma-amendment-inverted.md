# ADR-026 v2.1 — Amendement : formalisation du verdict `H7_SIGMA_INVERTED`

**Statut** : FROZEN (amendement à ADR-026 v2 §2)
**Date** : 2026-05-09 (post-main-run, pré-ADR-029)
**Portée** : amendement *uniquement procédural* — n'altère aucune constante de pré-enregistrement, n'altère aucune SHA pinnée du runner ou de la chaîne KL.

## 0. Pourquoi un amendement maintenant

Le main run sur le pool [1400-1429] a produit un signal massif (médiane δ_σ_corr = −0.148, Cohen d = −0.758, 26/30 seeds négatifs) dans la *direction opposée* à l'hypothèse pré-enregistrée H1-greater de ADR-027 §2.

ADR-026 v2 §2 incluait textuellement un bin :
> `d < 0 ∧ p < α → H7_SIGMA_INVERTED`

mais ce bin est **inatteignable** par construction : ADR-027 §3 fixe `alternative="greater"` ; quand δ_σ < 0, p_greater → 1 mécaniquement. Le verdict effectif retombe donc sur `INCONCLUSIVE` faute d'instrument capable d'asseoir l'inversion.

Cet amendement corrige un défaut formel pré-existant de v2 sans relâcher la rigueur statistique : le test miroir `alternative="less"` est ajouté, *avec son propre seuil α et seuil d*, **identiques au test primaire** (α=0.005, |d|≥0.5), et **sans nouveau degré de liberté** (pas de pêche : un seul test miroir, fixé à l'avance, sur la même statistique paired).

## 2. Correction Bonferroni : non requise

Le test `less` est *strictement complémentaire* du `greater` sur la même statistique paired :
- p_greater + p_less ≈ 1 (à la masse exacte près sur les zéros)
- les deux régions de rejet sont disjointes (`d ≥ +0.5` vs `d ≤ −0.5`)
- la décision est non-ambiguë : au plus un des deux peut rejeter

Donc le risque de type I global reste α = 0.005, sans Bonferroni. C'est strictement équivalent à pré-enregistrer un test bilatéral avec seuil α et statistique de Wilcoxon symétrisée — ce que ADR-027 §2 aurait dû faire dès le départ pour H7-σ comme classe d'hypothèse plutôt que comme prédiction directionnelle. Cette équivalence rend l'amendement *procédural* et non *théorique*.

## 3. Clause double-reporting (inchangée)

La clause "primary vs transparency" de ADR-026 v2 §2 s'applique à l'identique :
si le verdict du `delta_sigma_corr` ≠ verdict du `delta_sigma_naive` aux mêmes seuils
→ override `H7_SIGMA_INCONCLUSIVE`.
